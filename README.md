# Pong-Tracker-iOS
An iOS app to keep track of ping pong games, scores, and statistics.

## Installation

Pong-Tracker-iOS uses [CocoaPods](http://cocoapods.org). Make sure to perfom the 
following command before building the project:

```ruby
pod install
```
